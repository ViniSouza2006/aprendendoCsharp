O Csharp lê da Direita para esquerda

WRITE = Em C#, o método Write da classe Console
é usado para escrever texto no console sem adicionar uma nova linha no final.

WRITELINE = O método WriteLine da classe Console 
é usado para escrever texto no console e, em seguida, vai a próxima linha.

READLINE = O método ReadLine da classe Console 
é usado para ler o teclado (O que o usuario vai escrever).

INT.PARSE = O método int.Parse é usado para converter
uma string que representa um número em um valor do tipo int.

FLOAT.PARSE = Assim como int.Parse, o método float.Parse lançará uma exceção se a 
string não puder ser convertida para um número de ponto flutuante válido.
